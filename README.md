# Typing Speed Test - Day 3 Project

**By Noah Khomer**

A modern, dynamic typing speed test application inspired by the mountain hiking society theme. This project challenges users to type as fast and accurately as possible within a 1-minute timeframe while emphasizing the importance of upskilling in an AI-driven world.

### Core Functionality
- **1-Minute Timer**: Countdown timer with visual warnings (yellow at 20s, red at 10s)
- **Real-time WPM Calculation**: Words per minute calculation that updates dynamically
- **Accuracy Tracking**: Real-time accuracy percentage based on character comparison
- **Character Counter**: Live character count display
- **Multiple Text Samples**: 8 different inspirational and educational texts for variety

### User Experience
- **Modern UI Design**: Beautiful gradient backgrounds and glassmorphism effects
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices

  - `Escape` to reset test

- **Fixed Navigation Bar**: Professional navigation with smooth scrolling
- **Mobile-Friendly Menu**: Hamburger menu for mobile devices with smooth animations

## üéØ How to Use

1. **Navigate**: Use the top navigation bar to jump to different sections
2. **Start the Test**: Click the "Start Test" button or press `Ctrl/Cmd + Enter`
3. **Type the Text**: A random text will appear - type it as quickly and accurately as possible
4. **Monitor Progress**: Watch your real-time WPM, accuracy, and character count
5. **Complete the Test**: The test automatically ends after 1 minute or when you finish typing
6. **View Results**: See your final scores and take another test if desired

## üìä Metrics Explained

- **WPM (Words Per Minute)**: Number of words typed divided by time in minutes
- **Accuracy**: Percentage of correctly typed characters
- **Characters**: Total number of characters typed
- **Time**: Countdown from 60 seconds
